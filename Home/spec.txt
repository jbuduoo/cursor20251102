# 專案儀表板規格書

## 工具概述
專案儀表板是一個集中展示所有專案程式的入口頁面，按資料夾分類顯示所有程式，提供快速訪問和導航功能。

## 技術規格
- **檔案類型**: HTML5 單頁應用
- **語言**: HTML5, CSS3, JavaScript (ES6)
- **依賴**: 無（純前端實現）
- **瀏覽器支援**: 所有現代瀏覽器

## 功能特性

### 核心功能
1. **分類顯示**
   - 按資料夾分類（game、tools、story）
   - 每個分類有獨立的圖示和標題
   - 清晰的視覺區分

2. **程式卡片**
   - 卡片式設計
   - 顯示程式名稱、圖示、描述
   - 類型標籤改為按鈕樣式
   - 不顯示檔案路徑（已移除）
   - 點擊卡片開啟程式

3. **快速訪問**
   - 點擊卡片直接開啟對應程式
   - 新視窗開啟（target="_blank"）
   - 類型標籤按鈕不觸發連結（使用 stopPropagation）

4. **響應式設計**
   - 支援桌面和手機
   - 自適應網格佈局
   - 觸控友好

### 視覺設計
- 漸層紫色背景（#667eea 到 #764ba2）
- 白色分類容器
- 卡片式程式展示
- 類型標籤為按鈕樣式（可點擊，有懸停效果）
- 懸停動畫效果
- 清晰的視覺層次
- **頁頭高度**: 佔畫面10%（10vh）
- **緊湊設計**: 減少間距和內距，讓整體更精簡

## 操作說明

### 基本操作
1. **瀏覽程式**: 在主頁面瀏覽所有程式分類
2. **開啟程式**: 點擊程式卡片即可開啟對應程式（新視窗開啟）
3. **查看分類**: 按資料夾分類查看不同類型的程式
4. **類型標籤**: 點擊類型標籤按鈕不會觸發卡片連結

## 技術實現

### 資料結構
```javascript
const programs = {
    game: [...],    // 遊戲程式列表
    tools: [...],   // 工具程式列表
    story: [...]    // 故事檔案列表
}
```

### 卡片生成
- 動態生成程式卡片（使用 `<a>` 標籤）
- 根據分類渲染到對應容器
- 類型標籤使用按鈕元素（`<button>`）
- 按鈕文字統一顯示為「按鈕」
- 按鈕點擊使用 `stopPropagation` 阻止觸發卡片連結
- 不顯示檔案路徑
