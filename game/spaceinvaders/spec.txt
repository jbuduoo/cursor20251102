# 太空侵略者遊戲規格書

## 遊戲概述
太空侵略者是一款經典的射擊遊戲，玩家控制飛船射擊來襲的外星人，保護地球。

## 技術規格
- **檔案類型**: HTML5 Canvas 遊戲
- **語言**: HTML5, CSS3, JavaScript (ES6)
- **依賴**: 無（純前端實現）
- **瀏覽器支援**: 所有支援 Canvas 的現代瀏覽器

## 功能特性

### 核心功能
1. **玩家控制系統**
   - 左右移動
   - 射擊功能
   - 生命系統

2. **敵人系統**
   - 多排多列敵人陣列
   - 敵人移動模式（左右移動，觸邊下降）
   - 敵人自動射擊
   - 敵人被擊中後消失

3. **子彈系統**
   - 玩家子彈（向上）
   - 敵人子彈（向下）
   - 碰撞檢測

4. **計分系統**
   - 擊敗敵人得分
   - 生命值顯示
   - 遊戲結束判定

5. **視覺效果**
   - 星空背景
   - 彩色飛船和敵人
   - 發光效果

## 遊戲規則

### 基本規則
1. 使用左右方向鍵移動飛船
2. 使用空白鍵射擊
3. 擊敗所有敵人獲勝
4. 被敵人子彈擊中或敵人到達底部則失去生命
5. 生命耗盡遊戲結束

### 得分規則
- 擊敗一個敵人 = 10 分

## 操作說明

### 基本操作
1. **開始遊戲**: 點擊 "開始遊戲" 按鈕
2. **移動**: ⬅️ ➡️ 方向鍵
3. **射擊**: 空白鍵

## 技術實現

### 遊戲參數
- 畫布大小: 800x600 像素
- 敵人陣列: 5行 x 10列
- 玩家速度: 5 像素/幀
- 子彈速度: 7 像素/幀（玩家），3 像素/幀（敵人）
- 初始生命: 3

### 碰撞檢測
1. **子彈與敵人**: 檢查子彈是否與敵人重疊
2. **敵人子彈與玩家**: 檢查敵人子彈是否擊中玩家
3. **敵人與邊界**: 檢查敵人是否觸及邊界

### 敵人移動算法
- 水平移動，觸邊時反向並下降
- 速度隨時間增加
