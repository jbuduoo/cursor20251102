# 太空侵略者遊戲

經典的射擊遊戲，玩家控制飛船射擊來襲的外星人。

## 遊戲說明

太空侵略者是1978年發布的經典街機遊戲，玩家需要控制飛船左右移動，射擊來襲的外星人，保護地球不被入侵。

## 如何遊玩

1. 點擊 "開始遊戲" 按鈕
2. 使用 ⬅️ ➡️ 方向鍵移動飛船
3. 使用空白鍵射擊
4. 擊敗所有敵人獲勝
5. 避免被敵人子彈擊中

## 技術特點

- HTML5 Canvas 渲染
- 多敵人陣列系統
- 碰撞檢測系統
- 星空背景效果
