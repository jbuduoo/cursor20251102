# 打地鼠遊戲規格書

## 遊戲概述
打地鼠是一款經典的街機遊戲，玩家需要在時間內點擊出現的地鼠來得分。

## 技術規格
- **檔案類型**: HTML5 遊戲
- **語言**: HTML5, CSS3, JavaScript (ES6)
- **依賴**: 無（純前端實現）

## 功能特性
1. 9個洞的網格系統
2. 隨機出現地鼠
3. 計時系統（30秒）
4. 點擊得分系統
5. 視覺反饋效果

## 遊戲規則
- 點擊出現的地鼠得分
- 30秒內盡可能獲得高分
- 地鼠會自動消失

## 操作說明
- 點擊地鼠得分
- 點擊"開始遊戲"開始

## 版本資訊
- **版本**: 1.0
- **最後更新**: 2024
