# 小行星遊戲規格書

## 遊戲概述
小行星是一款經典的射擊遊戲，玩家控制飛船射擊小行星，避免碰撞。

## 技術規格
- **檔案類型**: HTML5 Canvas 遊戲
- **語言**: HTML5, CSS3, JavaScript (ES6)

## 功能特性
1. 飛船旋轉和移動控制
2. 射擊系統
3. 小行星物理系統
4. 碰撞檢測
5. 小行星分裂機制

## 遊戲規則
- 射擊小行星得分
- 撞到小行星失去生命
- 小行星被擊中後分裂
- 所有小行星消滅後生成新的

## 操作說明
- ⬆️ 加速 ⬅️ ➡️ 旋轉 空白鍵射擊

## 版本資訊
- **版本**: 1.0
