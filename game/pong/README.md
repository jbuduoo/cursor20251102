# Pong 遊戲

經典的乒乓球遊戲，玩家控制左側球拍與電腦對戰。

## 遊戲說明

Pong 是史上最早的電子遊戲之一，玩家需要控制球拍將球擊回對方區域，避免球從自己的一側出界。

## 如何遊玩

1. 使用 ⬆️ ⬇️ 方向鍵控制左側球拍
2. 將球擊回右側，避免球從左側出界
3. 球從右側出界時得分
4. 享受經典遊戲體驗！

## 技術特點

- HTML5 Canvas 渲染
- 簡單的 AI 對手
- 流暢的物理系統
- 響應式設計，滿屏顯示

## 檔案

- `pong.html` - 遊戲主檔案
- `spec.txt` - 遊戲規格說明
- `pong_log.txt` - 開發修改記錄

## 版面配置調整

- 遊戲頁面固定滿屏無捲軸，`body` 設定 `100vh` 與 `overflow: hidden`。
- 使用 Flex 置中標題、分數與畫布，標題／分數字體縮小以加大可視範圍。
- 透過 JavaScript 動態計算顯示尺寸，確保 Canvas 維持原始 2:1 長寬比並填滿可用空間。
