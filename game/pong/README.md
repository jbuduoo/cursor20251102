# Pong 遊戲

經典的乒乓球遊戲，玩家控制左側球拍與電腦對戰。

## 遊戲說明

Pong 是史上最早的電子遊戲之一，玩家需要控制球拍將球擊回對方區域，避免球從自己的一側出界。

## 如何遊玩

1. 使用 ⬆️ ⬇️ 方向鍵控制左側球拍
2. 將球擊回右側，避免球從左側出界
3. 球從右側出界時得分
4. 享受經典遊戲體驗！
