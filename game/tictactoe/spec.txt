# Tic-Tac-Toe 遊戲規格書

## 遊戲概述
Tic-Tac-Toe（井字遊戲）是一個經典的雙人對戰遊戲，玩家輪流在 3x3 的網格中放置 X 或 O，率先在橫、直或斜線連成三子的玩家獲勝。

## 技術規格
- **檔案類型**: HTML 單頁應用
- **語言**: HTML5, CSS3, JavaScript (ES6)
- **依賴**: 無（純前端實現）
- **瀏覽器支援**: 所有現代瀏覽器

## 功能特性

### 核心功能
1. **雙人對戰系統**
   - 玩家 X 和玩家 O 輪流下棋
   - 自動切換玩家
   - 顯示當前玩家狀態

2. **勝負判定**
   - 檢測橫向、直向、對角線的連線
   - 自動判定勝負
   - 高亮顯示獲勝的格子
   - 檢測平局情況

3. **計分系統**
   - 追蹤玩家 X 的勝利次數
   - 追蹤玩家 O 的勝利次數
   - 追蹤平局次數
   - 計分持久化（頁面刷新前）

4. **遊戲重置**
   - 一鍵重置遊戲
   - 清除所有棋子
   - 重置遊戲狀態

## 遊戲規則

### 基本規則
1. 遊戲在 3x3 的網格中進行
2. 玩家 X 先手
3. 玩家輪流在空白格子中放置標記
4. 不能覆蓋已放置的標記
5. 率先連成三子的玩家獲勝
6. 如果所有格子都被填滿且無人獲勝，則為平局

### 獲勝條件
- 橫向三連：第 1、2、3 行任一行
- 直向三連：第 1、2、3 列任一列
- 對角線三連：左上到右下 或 右上到左下

## 操作說明

### 基本操作
1. **下棋**: 點擊空白格子放置標記
2. **重新開始**: 點擊 "New Game" 按鈕
3. **查看分數**: 頂部顯示區域查看當前分數

### 鍵盤操作
- 無特殊鍵盤操作（純滑鼠點擊遊戲）

