# 井字遊戲 (Tic-Tac-Toe)

## 遊戲說明

Tic-Tac-Toe（井字遊戲）是一個經典的雙人對戰遊戲，玩家輪流在 3x3 的網格中放置 X 或 O，率先在橫、直或斜線連成三子的玩家獲勝。

## 檔案結構

- `tictactoe.htm` - 遊戲主檔案
- `tictactoe.md` - 遊戲規格書
- `tictactoe_log.txt` - 修改記錄

## 操作方式

1. **下棋**: 點擊空白格子放置標記
2. **重新開始**: 點擊 "New Game" 按鈕
3. **查看分數**: 頂部顯示區域查看當前分數

## 遊戲規則

- 遊戲在 3x3 的網格中進行
- 玩家 X 先手
- 玩家輪流在空白格子中放置標記
- 不能覆蓋已放置的標記
- 率先連成三子的玩家獲勝
- 如果所有格子都被填滿且無人獲勝，則為平局

## 獲勝條件

- 橫向三連：第 1、2、3 行任一行
- 直向三連：第 1、2、3 列任一列
- 對角線三連：左上到右下 或 右上到左下

## 技術需求

- 所有現代瀏覽器
- 無需額外依賴
