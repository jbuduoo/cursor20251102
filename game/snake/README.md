# 貪食蛇遊戲

## 遊戲說明

貪食蛇（Snake）是一款經典的街機風格遊戲，玩家控制一條蛇在網格中移動，吃食物來增長身體和得分，但要避免撞到牆壁或自己的身體。

## 檔案結構

- `snake.html` - 遊戲主檔案
- `snake.md` - 遊戲規格書
- `snake_log.txt` - 修改記錄

## 操作方式

1. **開始遊戲**: 按下任何方向鍵開始
2. **控制方向**: 
   - ⬆️ 上方向鍵 - 向上移動
   - ⬇️ 下方向鍵 - 向下移動
   - ⬅️ 左方向鍵 - 向左移動
   - ➡️ 右方向鍵 - 向右移動
3. **重新開始**: 點擊 "重新開始" 按鈕

## 遊戲規則

- 蛇在網格中自動移動
- 使用方向鍵控制移動方向
- 吃到紅色食物可以增長和得分
- 撞到牆壁或自己的身體會死亡
- 遊戲結束後可以重新開始

## 控制限制

- 不能立即反向移動（例如：向上時不能立即向下）
- 必須先按其他方向才能反向

## 技術需求

- 支援 Canvas 的現代瀏覽器
- 無需額外依賴
- 使用 localStorage 儲存最高分
