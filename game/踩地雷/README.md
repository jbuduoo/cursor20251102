# 踩地雷遊戲

一個完整的網頁版踩地雷遊戲，使用 HTML、CSS 和 JavaScript 開發。

## 遊戲特色

- 🎮 經典踩地雷玩法
- 📊 三種難度等級（簡單、中等、困難）
- ⏱️ 計時功能
- 🚩 右鍵標記地雷
- 💣 第一次點擊保證安全
- 🎨 現代化美觀界面

## 如何遊玩

1. 用瀏覽器打開 `index.html` 文件
2. 選擇難度等級（簡單/中等/困難）
3. 左鍵點擊格子揭示
4. 右鍵點擊格子標記地雷
5. 避免踩到地雷，標記所有地雷即獲勝！

## 遊戲規則

- 數字表示周圍 8 個格子中有幾個地雷
- 使用邏輯推理找出所有地雷的位置
- 標記所有地雷且揭示所有安全格子即獲勝
- 踩到地雷則遊戲結束

## 難度說明

- **簡單**: 9x9 格子，10 個地雷
- **中等**: 16x16 格子，40 個地雷
- **困難**: 16x30 格子，99 個地雷

祝您遊戲愉快！🎉
