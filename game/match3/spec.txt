# 消消樂遊戲規格書

## 遊戲概述
消消樂是一款三消益智遊戲，玩家交換相鄰方塊形成三個或更多相同顏色的連線來消除。

## 技術規格
- **檔案類型**: HTML5 遊戲
- **語言**: HTML5, CSS3, JavaScript (ES6)

## 功能特性
1. 8x8網格系統
2. 方塊交換機制
3. 三消檢測系統
4. 連鎖消除
5. 計分和移動限制

## 遊戲規則
- 點擊選擇方塊，再點擊相鄰方塊交換
- 形成三個或更多相同顏色連線消除
- 消除後上方方塊下落
- 30次移動內獲得最高分

## 操作說明
- 點擊選擇方塊
- 點擊相鄰方塊交換

## 版本資訊
- **版本**: 1.0
