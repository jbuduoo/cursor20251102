# 2048 遊戲

數字合併益智遊戲，通過滑動方塊合併相同數字。

## 如何遊玩
1. 使用方向鍵移動所有方塊
2. 相同數字方塊會合併
3. 每次移動後會出現新方塊
4. 目標是達到2048或更高

## 檔案
- `2048.html` - 遊戲主檔案
- `spec.txt` - 遊戲規格說明
- `2048_log.txt` - 開發修改記錄

## 版面配置調整
- 採用滿版無捲軸的版面，`body` 固定 `100vh` 並隱藏捲軸。
- 透過 Flex 置中資訊與棋盤，並縮小標題／分數字體以釋出空間。
- 以 JavaScript 動態計算棋盤長寬，保持正方形比例並填滿可用高度。
