# 2048 遊戲規格書

## 遊戲概述
2048是一款數字合併益智遊戲，玩家通過滑動方塊合併相同數字，目標是達到2048。

## 技術規格
- **檔案類型**: HTML5 遊戲
- **語言**: HTML5, CSS3, JavaScript (ES6)

## 功能特性
1. 4x4網格系統
2. 數字合併邏輯
3. 隨機生成新方塊
4. 計分系統
5. 最高分記錄（localStorage）

## 遊戲規則
- 使用方向鍵移動所有方塊
- 相同數字方塊碰撞時合併
- 每次移動後隨機生成新方塊（2或4）
- 無法移動時遊戲結束
- 達到2048獲勝（可繼續遊戲）

## 操作說明
- ⬆️ ⬇️ ⬅️ ➡️ 移動方塊

## 版本資訊
- **版本**: 1.0
