# 憤怒鳥遊戲

## 遊戲說明

憤怒鳥（Angry Birds）是一款物理模擬彈弓遊戲，玩家使用彈弓發射小鳥來摧毀建築結構，目標是清除所有磚塊或達到一定分數。

## 檔案結構

- `angrybirds.html` - 遊戲主檔案
- `angrybirds.md` - 遊戲規格書
- `angrybirds_log.txt` - 修改記錄

## 操作方式

1. **瞄準**: 將滑鼠移到小鳥上
2. **發射**: 
   - 按住滑鼠左鍵拖曳小鳥
   - 拖曳距離決定發射力度
   - 放開滑鼠左鍵發射
3. **查看資訊**: 頂部顯示得分、剩餘小鳥、關卡

## 遊戲規則

- 使用滑鼠拖曳小鳥來瞄準和發射
- 發射小鳥摧毀建築結構
- 清除所有磚塊或達到目標分數即可過關
- 有限的小鳥數量
- 小鳥用完且未完成目標則遊戲結束

## 技巧提示

- 瞄準結構的支撐點更容易摧毀
- 利用物理連鎖反應可以一次摧毀多個磚塊
- 觀察結構弱點，選擇最佳發射角度

## 技術需求

- 支援 Canvas 的現代瀏覽器
- 無需額外依賴
