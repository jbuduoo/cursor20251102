# 憤怒鳥遊戲規格書

## 遊戲概述
憤怒鳥（Angry Birds）是一款物理模擬彈弓遊戲，玩家使用彈弓發射小鳥來摧毀建築結構，目標是清除所有磚塊或達到一定分數。

## 技術規格
- **檔案類型**: HTML5 Canvas 物理模擬遊戲
- **語言**: HTML5, CSS3, JavaScript (ES6)
- **物理引擎**: 自定義 2D 物理引擎
- **依賴**: 無（純前端實現）
- **瀏覽器支援**: 所有支援 Canvas 的現代瀏覽器

## 遊戲規則

### 基本規則
1. 使用滑鼠拖曳小鳥來瞄準和發射
2. 發射小鳥摧毀建築結構
3. 清除所有磚塊或達到目標分數即可過關
4. 有限的小鳥數量
5. 小鳥用完且未完成目標則遊戲結束

### 物理規則
- 重力: 9.8 m/s²（模擬）
- 摩擦力: 影響物體移動
- 碰撞: 彈性碰撞反應
- 質量: 不同物體有不同的質量

### 得分規則
- 擊中磚塊獲得一定分數
- 連鎖反應摧毀額外加分

## 操作說明

### 基本操作
1. **瞄準**: 將滑鼠移到小鳥上
2. **發射**: 
   - 按住滑鼠左鍵拖曳小鳥
   - 拖曳距離決定發射力度
   - 放開滑鼠左鍵發射
3. **查看資訊**: 頂部顯示得分、剩餘小鳥、關卡

### 技巧提示
- 瞄準結構的支撐點更容易摧毀
- 利用物理連鎖反應可以一次摧毀多個磚塊
- 觀察結構弱點，選擇最佳發射角度
