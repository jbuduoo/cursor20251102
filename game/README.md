## 使用方式

1. 直接開啟各遊戲的 HTML 檔案即可遊玩
2. 查看各遊戲的 `.md` 檔案了解詳細規格
3. 查看各遊戲的 `_log.txt` 檔案了解修改歷史

## 技術規格

- **技術**: HTML5, CSS3, JavaScript (ES6)
- **渲染**: Canvas API
- **依賴**: 無（純前端實現）
- **瀏覽器**: 支援 Canvas 的現代瀏覽器

## 注意事項

- 所有遊戲都使用 Canvas 繪圖
- 部分遊戲使用 localStorage 儲存最高分
- 建議使用 Chrome 或 Edge 瀏覽器以獲得最佳體驗
