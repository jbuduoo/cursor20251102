# Flappy Bird 遊戲規格書

## 遊戲概述
Flappy Bird 是一款簡單但具有挑戰性的街機遊戲，玩家控制小鳥通過點擊或按鍵使其飛翔，避開管道障礙物，目標是獲得最高分數。

## 技術規格
- **檔案類型**: HTML5 Canvas 遊戲
- **語言**: HTML5, CSS3, JavaScript (ES6)
- **依賴**: 無（純前端實現）
- **瀏覽器支援**: 所有支援 Canvas 的現代瀏覽器

## 功能特性

### 核心功能
1. **小鳥飛行系統**
   - 重力系統影響小鳥下降
   - 點擊/按鍵產生向上推力
   - 平滑的飛行動畫
   - 旋轉效果（根據速度）

2. **管道系統**
   - 隨機高度的管道障礙
   - 上下管道配對
   - 管道間隙設計
   - 自動生成和移動

3. **碰撞檢測**
   - 小鳥與管道碰撞
   - 小鳥與上下邊界碰撞
   - 碰撞後遊戲結束

4. **計分系統**
   - 通過管道得分
   - 當前得分顯示
   - 最高分記錄（localStorage 持久化）
   - 自動更新最高分

5. **遊戲狀態管理**
   - 開始/結束狀態
   - 遊戲重置功能
   - 死亡動畫

### 視覺設計
- 漸層藍色背景（#4facfe 到 #00f2fe）
- 藍天風格畫布
- 黃色小鳥
- 綠色管道
- 白色雲朵裝飾

## 遊戲規則

### 基本規則
1. 小鳥會自動受重力影響下降
2. 點擊畫面、按空白鍵或方向鍵上使小鳥上升
3. 通過管道之間的間隙得分
4. 撞到管道或邊界會死亡
5. 遊戲結束後可以重新開始

### 得分規則
- 每通過一對管道 = 1 分
- 最高分會自動保存
- 分數在遊戲過程中即時更新

## 操作說明

### 基本操作
1. **開始遊戲**: 點擊畫面、按空白鍵或 ⬆️ 方向鍵
2. **控制飛行**: 
   - 點擊畫面
   - 按空白鍵
   - 按 ⬆️ 方向鍵
   - 以上操作都會使小鳥上升
3. **重新開始**: 點擊 "重新開始" 按鈕

### 遊戲技巧
- 保持節奏，不要過於頻繁點擊
- 觀察管道間隙位置
- 預判小鳥的飛行軌跡
- 保持冷靜，避免過度反應

## 技術實現

### 遊戲狀態管理
```javascript
- bird: 小鳥物件 {x, y, velocity, rotation}
- pipes: 管道陣列
- score: 當前得分
- bestScore: 最高分（localStorage）
- gameRunning: 遊戲運行狀態
- gravity: 重力常數
- jumpPower: 跳躍力度
```

### 物理系統
- **重力**: 持續向下加速
- **跳躍**: 點擊時給予向上速度
- **速度限制**: 防止速度過快
- **旋轉**: 根據速度計算旋轉角度

### 管道生成
- 固定間距生成新管道
- 隨機高度生成（確保可通過）
- 管道配對（上下管道）
- 移除超出畫布的管道

### 碰撞檢測
- 小鳥與管道矩形碰撞檢測
- 小鳥與畫布邊界檢測
- 精確的像素級碰撞

### 計分機制
- 管道通過檢測（小鳥 x 座標超過管道）
- 每個管道只計一次分
- 即時更新分數顯示
