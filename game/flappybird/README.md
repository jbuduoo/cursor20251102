# Flappy Bird 遊戲

## 遊戲說明

Flappy Bird 是一款簡單但具有挑戰性的街機遊戲，玩家控制小鳥通過點擊或按鍵使其飛翔，避開管道障礙物，目標是獲得最高分數。

## 檔案結構

- `flappybird.html` - 遊戲主檔案
- `flappybird.md` - 遊戲規格書
- `flappybird_log.txt` - 修改記錄

## 操作方式

1. **開始遊戲**: 點擊畫面、按空白鍵或 ⬆️ 方向鍵
2. **控制飛行**: 
   - 點擊畫面
   - 按空白鍵
   - 按 ⬆️ 方向鍵
   - 以上操作都會使小鳥上升
3. **重新開始**: 點擊 "重新開始" 按鈕

## 遊戲規則

- 小鳥會自動受重力影響下降
- 點擊畫面、按空白鍵或方向鍵上使小鳥上升
- 通過管道之間的間隙得分
- 撞到管道或邊界會死亡
- 遊戲結束後可以重新開始

## 遊戲技巧

- 保持節奏，不要過於頻繁點擊
- 觀察管道間隙位置
- 預判小鳥的飛行軌跡
- 保持冷靜，避免過度反應

## 技術需求

- 無需額外依賴
