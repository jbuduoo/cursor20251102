# 打磚塊遊戲

## 遊戲說明

打磚塊（Breakout）是一款經典的街機遊戲，玩家控制擋板反彈球來擊破上方的磚塊，目標是清除所有磚塊以過關。本版本支援多球系統和超高耐久度磚塊。

## 檔案結構

- `breakout.html` - 遊戲主檔案
- `breakout.md` - 遊戲規格書
- `breakout_log.txt` - 修改記錄

## 操作方式

1. **控制擋板**:
   - 滑鼠移動: 滑鼠在畫布上移動控制擋板
   - 鍵盤控制: ⬅️ 左方向鍵 / ➡️ 右方向鍵
2. **發射球**: 按空白鍵或點擊畫布發射未啟動的球
3. **查看資訊**: 頂部顯示得分、生命值、關卡

## 遊戲規則

- 使用滑鼠或方向鍵控制擋板
- 球反彈擊破磚塊
- 球掉落會扣除生命值
- 清除所有磚塊即可過關
- 生命值歸零遊戲結束

## 特殊機制

- **每秒生成新球**: 每 1 秒自動在擋板上生成新球
- **擋板變長**: 球掉落時擋板長度 +10 像素（最多到畫布寬度）
- **耐久度系統**: 磚塊需要多次擊中才會消失
- **難度遞增**: 支援 100 關，難度隨關卡增加

## 技術需求

- 無需額外依賴
