# 文字轉語音工具

## 工具說明

文字轉語音工具是一個基於 Web Speech API 的語音合成工具，可以將輸入的文字轉換為語音並播放，支援多種語音選項和參數調整。

## 檔案結構

- `text-to-speech.html` - 工具主檔案
- `text-to-speech.md` - 工具規格書
- `text-to-speech_log.txt` - 修改記錄

## 主要功能

1. **文字轉語音** - 使用 Web Speech API 將文字轉換為語音
2. **語音控制** - 播放/停止控制，語速、音調、音量調整
3. **語音選擇** - 支援多種語音，自動偵測可用語音
4. **狀態顯示** - 播放狀態指示，當前設定顯示

## 操作方式

1. **輸入文字**: 在文字框中輸入要轉換的文字
2. **選擇語音**: 從下拉選單中選擇語音
3. **調整參數**: 調整語速、音調、音量滑桿
4. **播放**: 點擊「播放」按鈕開始播放
5. **停止**: 點擊「停止」按鈕停止播放

## 參數說明

- **語速**: 控制語音播放速度（0.5x - 2.0x）
- **音調**: 控制語音音調高低（0.5 - 2.0）
- **音量**: 控制語音音量大小（0 - 1）

## 系統需求

- **瀏覽器**: 所有現代瀏覽器（Chrome, Edge, Safari, Firefox）
- **權限**: 無需特殊權限
- **網路**: 無需網路連線（本地語音合成）

## 注意事項

1. 不同瀏覽器支援的語音可能不同
2. 某些語音可能需要下載
3. 語音品質取決於系統和瀏覽器
