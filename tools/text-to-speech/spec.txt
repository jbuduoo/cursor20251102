# 文字轉語音工具規格書

## 工具概述
文字轉語音工具是一個基於 Web Speech API 的語音合成工具，可以將輸入的文字轉換為語音並播放，支援多種語音選項和參數調整。

## 技術規格
- **檔案類型**: HTML5 單頁應用
- **語言**: HTML5, CSS3, JavaScript (ES6)
- **API**: Web Speech API (SpeechSynthesis)
- **依賴**: 無（純前端實現）
- **瀏覽器支援**: 所有現代瀏覽器

## 功能特性

### 核心功能
1. **文字轉語音**
   - 使用 Web Speech API
   - 支援多種語音選項
   - 可調整語速、音調、音量
   - 支援多種語言

2. **語音控制**
   - 播放/停止控制
   - 語速調整（0.5x - 2.0x）
   - 音調調整（0.5 - 2.0）
   - 音量調整（0 - 1）

3. **語音選擇**
   - 支援多種語音
   - 自動偵測可用語音
   - 語音列表顯示

4. **狀態顯示**
   - 播放狀態指示
   - 當前設定顯示
   - 操作提示

### 視覺設計
- 漸層紫色背景（#667eea 到 #764ba2）
- 白色容器，圓角設計
- 響應式控制介面
- 狀態指示器

## 操作說明

### 基本操作
1. **輸入文字**: 在文字框中輸入要轉換的文字
2. **選擇語音**: 從下拉選單中選擇語音
3. **調整參數**: 調整語速、音調、音量滑桿
4. **播放**: 點擊「播放」按鈕開始播放
5. **停止**: 點擊「停止」按鈕停止播放

### 參數說明
- **語速**: 控制語音播放速度
- **音調**: 控制語音音調高低
- **音量**: 控制語音音量大小
