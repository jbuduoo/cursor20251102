# 互動活動管理系統

## 工具說明

互動活動管理系統是一個純前端的即時問答與回饋收集工具，支援主持人建立活動、參與者即時參與，無需後端伺服器。使用 localStorage 儲存數據，透過 URL 參數共享房間 ID 實現活動連接。

## 檔案結構

```
interactive-activity/
├── spec.txt                    # 規格書
├── README.md                   # 使用說明（本檔案）
├── interactive-activity_log.txt # 修改記錄
├── activity-manager.js        # 活動管理邏輯（共用）
├── host-dashboard.html        # 主持人儀表板（主頁）
├── host-game.html             # 主持遊戲頁面
└── participant.html           # 參與者頁面
```

## 主要功能

### 1. 即時問答（Q&A）
- 主持人建立選擇題問答活動
- 參與者選擇答案並即時提交
- 即時顯示各選項統計
- 即時排行榜（前 10 名）
- 自動計分系統

### 2. 即時回饋（Feedback）
- 主持人建立匿名回饋收集活動
- 參與者匿名提交意見或問題
- 主持人可標記已讀/未讀
- 主持人可刪除回饋

## 使用方式

### 主持人端

1. **建立活動**
   - 開啟 `host-dashboard.html`
   - 點擊「建立問答」或「建立回饋」
   - 填寫活動資訊
   - 系統會生成房間 ID（如：`ABC123`）

2. **主持活動**
   - 在活動列表中點擊「主持遊戲」
   - 進入 `host-game.html?room=ABC123`
   - 分享參與者連結給觀眾

3. **管理活動**
   - 編輯活動：點擊「編輯」按鈕（目前為簡化版，會刪除並重新建立）
   - 刪除活動：點擊「刪除」按鈕

### 參與者端

1. **加入活動**
   - 開啟參與者連結：`participant.html?room=ABC123`
   - 輸入名稱（可選）
   - 開始參與活動

2. **問答模式**
   - 查看當前問題
   - 選擇答案（只能選擇一次）
   - 查看即時排行榜和自己的排名

3. **回饋模式**
   - 輸入回饋內容
   - 點擊「提交回饋」
   - 查看所有回饋（匿名顯示）

## 技術規格

- **技術**: HTML5, CSS3, JavaScript (ES6)
- **數據儲存**: localStorage
- **同步機制**: 輪詢（每 2 秒）
- **瀏覽器支援**: 現代瀏覽器（Chrome, Edge, Safari, Firefox）
- **依賴**: 無（純前端實現）

## 系統需求

- 現代瀏覽器（支援 localStorage）
- 不需要網路連線（單機使用）
- 不需要後端伺服器

## 注意事項

### 限制

1. **單機使用**
   - 數據儲存在瀏覽器的 localStorage 中
   - 僅限同一瀏覽器或需要手動分享 URL
   - 清除瀏覽器數據會遺失所有活動

2. **同步延遲**
   - 參與者頁面每 2 秒自動同步一次
   - 不是真正的即時同步

3. **跨設備限制**
   - 不支援跨設備即時同步
   - 如需跨設備使用，需要手動分享 URL

### 使用建議

1. **房間 ID 分享**
   - 主持人建立活動後，記下房間 ID
   - 將參與者連結分享給觀眾
   - 參與者連結格式：`participant.html?room=房間ID`

2. **數據備份**
   - 重要活動建議定期備份
   - 可透過瀏覽器開發者工具匯出 localStorage 數據

3. **活動管理**
   - 活動結束後建議刪除，避免數據累積
   - 編輯功能目前為簡化版，會刪除並重新建立

## 操作流程範例

### 問答活動流程

1. 主持人開啟 `host-dashboard.html`
2. 點擊「建立問答」
3. 填寫活動標題：例如「AI知識問答」
4. 新增問題：
   - 問題：什麼是 AI？
   - 選項 A：人工智慧
   - 選項 B：自動化系統
   - 選項 C：機器學習
   - 選項 D：深度學習
   - 正確答案：選項 A
5. 點擊「建立」，記下房間 ID（如：`ABC123`）
6. 點擊「主持遊戲」，進入主持頁面
7. 分享連結：`participant.html?room=ABC123`
8. 參與者開啟連結，選擇答案
9. 主持人查看即時統計和排行榜
10. 活動結束後點擊「結束活動」

### 回饋活動流程

1. 主持人開啟 `host-dashboard.html`
2. 點擊「建立回饋」
3. 填寫活動標題：例如「意見回饋收集」
4. 填寫回饋提示文字（可選）
5. 點擊「建立」，記下房間 ID
6. 點擊「主持遊戲」，進入主持頁面
7. 分享參與者連結
8. 參與者提交回饋
9. 主持人查看和管理回饋
10. 活動結束後點擊「結束活動」

## 常見問題

### Q: 參與者看不到最新狀態？
A: 參與者頁面每 2 秒自動同步，請稍候。也可以手動重新整理頁面。

### Q: 活動數據會遺失嗎？
A: 數據儲存在瀏覽器的 localStorage 中，清除瀏覽器數據會遺失。建議定期備份重要活動。

### Q: 可以多人同時參與嗎？
A: 可以，但需要所有參與者使用相同的瀏覽器或手動分享 URL。不支援跨設備即時同步。

### Q: 如何匯出活動結果？
A: 目前不支援匯出功能，可透過瀏覽器開發者工具查看 localStorage 數據。
