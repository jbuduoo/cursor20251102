# 語音轉文字工具

## 工具說明

語音轉文字工具是一個基於 Web Speech API 的語音識別工具，可以將使用者的語音即時轉換為文字，並提供多種文字處理功能。

## 檔案結構

- `speech-to-text.html` - 工具主檔案
- `speech-to-text.md` - 工具規格書
- `speech-to-text_log.txt` - 修改記錄

## 主要功能

1. **語音識別** - 持續語音識別，即時顯示識別結果
2. **AI 順稿** - 自動改善文字格式，調整標點符號
3. **語音播放** - 文字轉語音功能
4. **翻譯成英語** - 將繁體中文翻譯成英語
5. **AI 指令** - 將文字格式化為 `claude -p "文字"` 格式
6. **特殊功能** - 大猩猩、一直笑、罵髒話等趣味功能

## 操作方式

1. **開始錄音**: 點擊「🎤 開始錄音」按鈕
2. **停止錄音**: 點擊「⏹ 停止錄音」按鈕
3. **清除文字**: 點擊「🗑 清除文字」按鈕
4. **AI 順稿**: 點擊「✏️ AI 順稿」按鈕改善文字
5. **語音播放**: 點擊「🔊 語音播放」按鈕朗讀文字
6. **翻譯成英語**: 點擊「🌐 翻譯成英語」按鈕翻譯文字
7. **AI 指令**: 點擊「🤖 AI 指令」按鈕格式化並複製指令

## 系統需求

- **瀏覽器**: Chrome 或 Edge（需支援語音識別 API）
- **權限**: 需要麥克風權限
- **網路**: 需要網路連線（語音識別和翻譯需要伺服器）

## 注意事項

1. 需要網路連線（語音識別和翻譯需要伺服器）
2. 需要麥克風權限
3. 建議在安靜環境中使用
4. 翻譯功能依賴第三方 API，可能會有使用限制
