# 音樂工具集規格書

## 專案名稱
音樂工具集 (Music Tools)

## 專案類型
工具類網頁應用程式

## 開發日期
2025-01-07

## 功能需求

### 1. 節拍器 (Metronome)
- BPM 調整範圍：40-200
- 視覺化節拍指示器
- 聲音節拍提示
- 開始/停止控制
- 快速調整按鈕（±5 BPM）

### 2. 簡易鋼琴 (Simple Piano)
- 一個八度的鍵盤（C4 到 C5）
- 包含白鍵和黑鍵
- 點擊播放對應音符
- 使用 Web Audio API

### 3. 音高檢測器 (Pitch Detector)
- 麥克風輸入檢測
- 顯示頻率（Hz）
- 顯示對應音符名稱
- 即時更新

### 4. BPM 檢測器 (BPM Tap)
- 點擊打拍子
- 計算平均 BPM
- 重置功能

## 技術規格

### 前端技術
- HTML5
- CSS3
- JavaScript (ES6+)

### API 使用
- Web Audio API
- MediaDevices API (getUserMedia)

### 瀏覽器要求
- 支援 Web Audio API
- 支援 getUserMedia API
- 現代瀏覽器（Chrome, Edge, Firefox, Safari）

## UI/UX 設計

### 佈局
- 響應式網格佈局
- 卡片式設計
- 漸層背景

### 顏色主題
- 主色：紫色漸層 (#667eea 到 #764ba2)
- 卡片：白色背景
- 按鈕：主色系

## 開發注意事項

1. 所有功能使用純前端技術實現
2. 不依賴外部函式庫
3. 需要處理瀏覽器相容性
4. 麥克風權限需要用戶授權
5. Web Audio API 需要用戶互動後才能啟動
