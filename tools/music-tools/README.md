# 音樂工具集

一個集合多種實用音樂工具的網頁應用程式，使用純 HTML、CSS 和 JavaScript 開發。

## 功能介紹

### 1. 節拍器 🥁
- 可調整 BPM（每分鐘節拍數）範圍：40-200
- 視覺化節拍指示器
- 聲音節拍提示
- 支援快速調整 BPM（±5）

### 2. 簡易鋼琴 🎹
- 一個八度的鋼琴鍵盤（C4 到 C5）
- 包含白鍵和黑鍵
- 點擊鍵盤即可播放對應音符
- 使用 Web Audio API 產生聲音

### 3. 音高檢測器 🎤
- 使用麥克風輸入檢測音高
- 顯示頻率（Hz）和對應的音符名稱
- 即時更新顯示
- 需要瀏覽器麥克風權限

### 4. BPM 檢測器 ⏱️
- 點擊打拍子測量 BPM
- 自動計算平均節拍速度
- 最多記錄 10 次點擊
- 可重置重新測量

## 使用方式

1. 直接在瀏覽器中開啟 `music-tools.html`
2. 選擇需要的工具使用
3. 音高檢測器需要允許瀏覽器存取麥克風

## 技術規格

- **技術**: HTML5, CSS3, JavaScript (ES6)
- **API**: Web Audio API, MediaDevices API
- **依賴**: 無（純前端實現）
- **瀏覽器**: 支援 Web Audio API 的現代瀏覽器（Chrome, Edge, Firefox, Safari）

## 注意事項

- 音高檢測器需要麥克風權限
- 建議使用 Chrome 或 Edge 瀏覽器以獲得最佳體驗
- 節拍器和鋼琴使用 Web Audio API，某些瀏覽器可能需要用戶互動後才能播放聲音

## 瀏覽器相容性

- ✅ Chrome/Edge (推薦)
- ✅ Firefox
- ✅ Safari
- ⚠️ 舊版瀏覽器可能不支援部分功能
