# AI 框架控制面板

一個視覺化介面工具，用於選擇網頁框架、設定尺寸比例和定義風格，最終生成可用於其他 AI 網頁工具的精確設計提示詞。

## 功能特點

- 🎨 **視覺化框架選擇**：四個預設框架佈局（左側邊欄、右側邊欄、三欄式、全寬式）
- 📐 **精確參數設定**：頁頭高度、側邊欄比例、風格描述等
- ✨ **智能提示詞生成**：自動生成完整的設計規格提示詞
- 📋 **一鍵複製**：快速複製生成的提示詞到剪貼板
- 📱 **完全響應式**：適配桌面、平板和手機

## 使用說明

### 1. 選擇框架

在左側預覽區點擊四個框架卡片之一：
- **左側邊欄**：側邊欄在左側，主內容在右側
- **右側邊欄**：主內容在左側，側邊欄在右側
- **三欄式**：左側邊欄、主內容、右側邊欄
- **全寬式**：單欄全寬佈局

選中的框架會有藍色邊框和背景高亮。

### 2. 設定參數

在右側面板填寫以下參數：

- **頁頭高度**：輸入百分比（0-100）
- **側邊欄寬度比例**：
  - 三欄式：格式如 `1.5:7:1.5`（左:中:右）
  - 兩欄式：格式如 `1.5:7` 或 `7:1.5`
- **網站用途/行業**：例如「企業官網」、「電商平台」、「部落格」
- **核心風格描述**：例如「現代極簡」、「專業商務」、「清新活潑」
- **間距控制**：單選一個選項
  - 高留白
  - 適中留白
  - 緊湊排版

### 3. 生成提示詞

點擊「✨ 生成提示詞」按鈕，系統會根據您的設定生成完整的設計提示詞。

### 4. 複製提示詞

生成後，點擊「📋 複製提示詞」按鈕，提示詞會自動複製到剪貼板，可直接用於其他 AI 工具。

## 設計風格

- **核心風格**：現代極簡、功能導向、清晰直觀
- **配色方案**：
  - 主色：淺灰色系（#F8F8F8 背景，#E0E0E0 邊框）
  - 強調色：專業藍色（#007BFF）
  - 文字：深灰色（#333333）
- **字體**：簡潔的無襯線字體（Arial, Helvetica Neue, Noto Sans）
- **間距**：適中到高留白，確保各元素有足夠的呼吸空間

## 響應式設計

- **桌面**：兩欄式佈局（左 60%，右 40%）
- **平板**：垂直堆疊，保持兩欄框架網格
- **手機**：單欄垂直堆疊，所有控制項易於訪問
